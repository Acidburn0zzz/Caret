    Bienvenue sur GitHub Pages.

    Ce générateur automatique de page est la meilleure façon de créer de belles pages pour tous vos projets. Auteur contenu de votre page ici en utilisant GitHub Flavored Markdown, sélectionnez un modèle conçu par un designer, et de publier. Après votre page est généré, vous pouvez consulter la nouvelle branche gh-pages localement. Si vous utilisez GitHub Desktop, synchroniser simplement votre dépôt et vous verrez la nouvelle branche.

    modèles de créateurs

    Nous avons élaboré des modèles beaux pour vous d'utiliser. Allez-y et cliquez sur "Continuer à layouts 'pour naviguer à travers eux. Vous pouvez facilement revenir en arrière pour modifier votre page avant de publier. Après la publication de votre page, vous pouvez revoir le générateur de page et passer à un autre thème. Le contenu de votre page sera préservée.

    Création de pages manuellement

    Si vous préférez ne pas utiliser le générateur automatique, pousser une branche nommée gh-pages à votre référentiel pour créer une page manuellement. En plus de soutenir le contenu HTML normale, Pages GitHub soutenir Jekyll, un blog conscient générateur de site simple, statique. Jekyll, il est facile de créer des en-têtes et pieds de page l'ensemble du site sans avoir à les copier sur chaque page. Il offre également un soutien de blog intelligent et d'autres fonctionnalités de texturants avancées.
